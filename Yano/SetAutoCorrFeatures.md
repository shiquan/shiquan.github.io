

# SetAutoCorrFeatures

[**Source code**](https://github.com/shiquan/Yano/tree/master/R/#L)

## Description

Set autocorrection by Moran’s I and/or p value.

## Usage

<pre><code class='language-R'>SetAutoCorrFeatures(
  object = NULL,
  moransi.min = 0,
  assay = DefaultAssay(object),
  p.thresh = 0.01,
  prefix = "moransi",
  verbose = TRUE
)
</code></pre>
